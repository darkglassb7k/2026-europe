# 古典文藝復興 Renaissance — 島嶼共鳴 2026

## Style Philosophy

文藝復興設計哲學以「**人文、秩序、古典對稱**」為核心——承襲 15-16 世紀義大利佛羅倫斯的人文主義覺醒，講究黃金比例與數學的和諧。視覺上：以**羊皮紙白、古籍墨藍、大理石灰**為基調，點綴**佛羅倫斯磚紅與溫潤的聖金**，展現如達文西手稿般的知性與優雅。在音樂節網頁中，這風格把「島嶼共鳴」呈現得像 **托斯卡尼莊園中的古典樂章**：均衡、精緻、富有文化底蘊，華麗而不流於浮誇。

三個視覺辨識特徵：
1. **羊皮紙白 + 墨藍 + 聖金 / 磚紅** 配色，沉穩且具人文厚度
2. **古典襯線體 + 絕對對稱排版 + 嚴謹直角或微圓角（0-4px）**
3. **達文西風幾何線條與古典拱門** 作為結構點綴
